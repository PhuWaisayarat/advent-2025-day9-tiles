# Advent of Code 2025 Day 9 Tiles Solving KB

This is a currently-in progress Prolog KB for solving the puzzle in Advent of Code 2025 Day 9. It can now currently solve the first part, but the second part is still not finished.

The predicates are partially documented in tiles_documentation.prolog. Try `doc_topic(Topic).`, `documentation(assertz_red_tiles).` and `documentation(part_1_solution).`.
